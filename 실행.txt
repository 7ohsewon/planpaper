1. 가상환경 설정 및 활성화 (선택 사항이지만 권장)
프로젝트 폴더 내에서 터미널을 열고 가상환경을 생성하고 활성화합니다.

bash
# 가상환경 생성
python -m venv venv
# 가상환경 활성화 (Windows PowerShell 기준)
.\venv\Scripts\Activate.ps1
# 가상환경 활성화 (Windows CMD 기준)
.\venv\Scripts\activate.bat

2. 패키지 설치
requirements.txt에 기재된 의존성 패키지들(Streamlit, OpenAI/Anthropic SDK, 파일 추출 라이브러리 등)을 설치합니다.

bash
pip install -r requirements.txt
3. Streamlit 애플리케이션 실행
설치가 완료되면 다음 명령어로 Streamlit 웹 애플리케이션을 실행합니다.

bash
streamlit run app.py
실행 후 브라우저에서 http://localhost:8501 (혹은 터미널에 표시되는 주소 및 포트)로 접속하여 사용하실 수 있습니다.
